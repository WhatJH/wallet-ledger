# 가계부 (Wallet Ledger)

**"광고 없이 깔끔하고, 나만 쓸 수 있는 가벼운 가계부"**

## 🔗 배포 링크
**[가계부 바로가기](https://wallet-ledger-zeta.vercel.app/)**

## 💡 프로젝트 소개
기존의 가계부 앱들은 사용하려고 하면 중간중간 뜨는 **불필요한 광고** 때문에 사용하기 불편했습니다.

그래서 **"광고 없이 쾌적하게"**, 그리고 **"군더더기 없는 핵심 기능만"** 담은 나만의 가계부를 직접 제작하게 되었습니다. 

## ✨ 주요 특징
- **No Ads**: 사용자의 집중력을 흐리는 광고가 전혀 없습니다.
- **Privacy First**: 로그인 없이도 브라우저 고유 ID(`localStorage`)를 통해 기기별로 독립된 데이터를 안전하게 보관합니다.
- **Clean UI**: 직관적인 달력 UI와 일별 상세 내역 확인 기능을 제공합니다.
- **Mobile Friendly**: PWA처럼 홈 화면에 추가하여 설치형 앱처럼 편리하게 사용할 수 있습니다.

## 🛠 Tech Stack
- **Frontend**: React, Vite, Tailwind CSS
- **Backend/Database**: Supabase
- **Deployment**: Vercel
